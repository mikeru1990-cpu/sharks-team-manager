# Full UX Migration Sprint

## Objective
Transform Sharks Team Manager from a dashboard-style web app into a unified football operating platform with native-feeling navigation, operational density, and live matchday flow.

---

# Core Architecture

## PremiumAppShell
Persistent native app shell with:
- Bottom navigation
- Independent screen identity
- Native-feeling layout flow
- Isolated app sections

## OperationalScreenHeader
Compact sticky headers replacing legacy hero sections.

## OperationalCard
Unified operational card architecture.

## NativeTabTransition
Smooth tab transitions with scroll reset.

## HomeMissionControl
Operational-first home experience.
